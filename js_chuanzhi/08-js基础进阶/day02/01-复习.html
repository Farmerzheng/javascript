js三大构成：
    1. ECMAScript   JS语言的标准
    2. DOM操作（DOM API）  ----> 查找元素，删除元素，修改元素，事件
    3. BOM( 浏览器相关的信息，关闭，打开，前进，后退 )

    书写位置：
    1.内嵌式
    2. 外链式 ，外部新建一个js文件

    三种输出方式
    弹出警示窗口   alert(123)
    输出到页面     document.write(123)
    输出到控制台   console.log(123)

    变量
    创建变量   var a = 1;
    变量名称规定： 以字母、$、_,后边可以字母，数字，下划线，$；中文
    一些保留字，或者关键字不能作为变量名称。
    var console = 1;
    var var = 2;

    //交换变量
    var a = 1;
    var b = 2;
    var tmp = a;
    a = b;
    b = tmp;

    数值型  1  2  2.3   5e+2        number
    字符串  'a' '1' 'hello'  "111"  string
    布尔型  true  false             boolean
    对象型  null                    object
    未定义型                        undefined
